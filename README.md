# Agriculture Skills

设施农业领域知识文档集，涵盖温室作物监测的数据标准、光照诊断、采收决策等核心环节。

> 这些文档定义的是**通用的农业领域知识**（数据结构、阈值参数、决策规则），不依赖特定平台或框架，任何智能体或项目均可直接使用。

## Skills 列表

### 🌿 agri-data-standard
**设施农业数据标准格式规范 v1.0.0**

定义了从图像采集到智能体决策全链路的数据结构，涵盖：
- 地块管理（Plot）与批次管理（Batch）
- 图像感知与目标检测结果
- 场景摘要与诊断结果
- 决策建议与历史记忆
- 适用于草莓、番茄、蓝莓、黄瓜等设施栽培作物

### 💡 greenhouse-light-diagnosis
**温室作物光照诊断与数据可靠性评估 v1.0.0**

基于温室环境感知数据，诊断数据可靠性和光照相关问题：
- 光照强度阈值分析（低光 / 适光 / 过曝）
- 阴影覆盖检测与置信度评估
- 历史趋势对比
- 生成诊断建议与风险提醒

### 🍓 strawberry-harvest-decision
**草莓成熟度评估与采收决策 v1.0.0**

基于感知数据和诊断结果，生成可操作的农业建议：
- 成熟度阈值判定（采收 ≥ 0.80 / 观察 ≥ 0.60）
- 可信度检查与历史趋势佐证
- 补光、人工复核等操作建议

## 使用方式

### 作为 AI 智能体的领域知识
直接将 SKILL.md 文件作为知识库注入即可，适用于：
- YouClaw、Dify、Coze、FastGPT 等 AI 平台
- 基于 LLM 的自定义 Agent 系统
- RAG（检索增强生成）知识库

### 作为项目开发参考
参考其中的数据结构定义和参数阈值：
- 农业物联网项目的数据模型设计
- 温室监测系统的决策逻辑实现
- YOLO 等视觉检测结果的后续处理流水线

## 数据来源

本 Skills 的内容来源于农业物联网与计算机视觉相关的科研实践，包括 YOLOv8 目标检测、PINN 物理信息神经网络等技术的温室应用经验。
