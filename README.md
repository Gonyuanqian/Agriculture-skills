# 农业领域 YouClaw Skills

适用于 [YouClaw](https://youclaw.com) 桌面 AI 助手的设施农业技能包。

## Skills 列表

### 🌿 agri-data-standard
**设施农业数据标准格式规范 v1.0.0**

定义了从图像采集到智能体决策全链路的数据结构，涵盖：
- 地块管理（Plot）与批次管理（Batch）
- 图像感知与目标检测结果
- 场景摘要与诊断结果
- 决策建议与历史记忆
- 适用于草莓、番茄、蓝莓、黄瓜等设施栽培作物

### 💡 greenhouse-light-diagnosis
**温室作物光照诊断与数据可靠性评估 v1.0.0**

基于温室环境感知数据，诊断数据可靠性和光照相关问题：
- 光照强度阈值分析（低光/适光/过曝）
- 阴影覆盖检测与置信度评估
- 历史趋势对比
- 生成诊断建议与风险提醒

### 🍓 strawberry-harvest-decision
**草莓成熟度评估与采收决策 v1.0.0**

基于感知数据和诊断结果，生成可操作的农业建议：
- 成熟度阈值判定（采收 ≥ 0.80 / 观察 ≥ 0.60）
- 可信度检查与历史趋势佐证
- 补光、人工复核等操作建议

## 安装方法

将 Skill 文件夹复制到 YouClaw 的 skills 目录下即可自动识别：

```
%USERPROFILE%\.youclaw\skills\
```

然后在 YouClaw 的「技能」模块中启用即可。

## 数据来源

本项目 Skills 的内容来源于农业物联网与计算机视觉相关的科研实践，包括 YOLOv8 目标检测、PINN 物理信息神经网络等技术的温室应用经验。
